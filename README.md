# Kermit
## Audio Controls
"S" = Start Recording

"P" = Stop Recording (Pause)

"=" = preAmp +

"-" = preAmp -

"[" = Quality -

"]" = Quality +

"Num +" = Noisegate +

"Num -" = Noisegate -
